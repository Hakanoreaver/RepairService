Instructions To Run

Pre Requisites
--
Java 8 plus
MySQL
Web Server on port 7080

Instructions
--
1. Create a mysql table databasebackend
2. Create a mysql user named 'user' identified by 'password'
3. Run the run.jar using java -jar run.jar from the terminal
4. Run your web server we use web server for chrome and set the home folder to Front End
5. Open the web server in browser and select main.html to begin using the program.
